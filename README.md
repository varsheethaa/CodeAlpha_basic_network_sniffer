# CodeAlpha_basic_network_sniffer
This project is a basic network packet sniffer developed to monitor and capture network traffic in real time. It listens to network interfaces and displays essential packet information such as source IP, destination IP, protocol type, and packet size.
 Basic Network Sniffer

A Basic Network Sniffer is a simple tool designed to capture and analyze network packets in real time. This project helps beginners understand how data is transmitted across a network and introduces the fundamentals of packet sniffing and network security.

 Features

Captures live network traffic

Displays source and destination IP addresses

Identifies protocols such as TCP, UDP, and ICMP

Shows packet length and basic header information

Lightweight and easy to use

Technologies Used

Programming Language: Python

Libraries: Socket / Scapy

Platform: Windows / Linux

 How It Works

The program listens to a selected network interface.

Incoming packets are captured in real time.

Packet headers are analyzed to extract key details.

The captured information is displayed to the user.
